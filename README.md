# Japan Tour Landing Page 🇯🇵

A modern and responsive Japan travel landing page built with React, Vite, and Tailwind CSS.

## ✨ Features

* Beautiful modern UI
* Fully responsive design
* Fast performance with Vite
* Tailwind CSS styling
* Interactive sections
* Travel-themed landing experience

## 🛠️ Tech Stack

* React
* Vite
* Tailwind CSS
* TypeScript

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/binitnayak/japan-tour.git
```

Go to project folder:

```bash
cd japan-tour
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

## 🚀 Build for Production

```bash
npm run build
```

## 📁 Project Structure

* `src/` → React components & pages
* `public/` → Static assets
* `package.json` → Dependencies and scripts
