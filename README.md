# ✨ Overview

Japan Tour is a beautifully designed landing page that showcases the culture, destinations, and travel experience of Japan.

The website combines cinematic visuals, elegant typography, interactive sections, and smooth scrolling animations to create an engaging user experience.

Whether you're exploring Mount Fuji, Tokyo, Kyoto, or Osaka, this project delivers a premium travel-inspired interface.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/binitnayak/japan-tour.git
```

Move into the project

```bash
cd japan-tour
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Open

```
http://localhost:5173
```

---

# 📦 Build

```bash
npm run build
```

Preview

```bash
npm run preview
```
